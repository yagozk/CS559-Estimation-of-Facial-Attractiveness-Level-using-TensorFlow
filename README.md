## Repository Contents
- `Best_VGG_Dilated.ipynb` – Final dilated-convolution VGG variant that achieved the best validation scores.
- `Best_VGG_MAX.ipynb` – Best performing baseline that keeps standard VGG max-pooling blocks.
- `Dilated_VGG_FullyConvolutional.ipynb` – Dilated VGG backbone augmented with fully convolutional prediction head experiments.
- `DilatedVGG_Dense.ipynb` – Early prototype of the dilated VGG + dense head pipeline, kept for reference on hyperparameters.
- `MAXVGG_Dense.ipynb` – Standard max-pooling VGG coupled with a dense regressor for attractiveness estimation.
- `MAXVGG_FullyConvolutional.ipynb` – Fully convolutional adaptation of the max-pooling VGG variant for end-to-end score maps.
